# Code repository of the warm up project of the university course Coding Camp 1 - Python in german UDS

## Extension of the Django Polls tutorial 

This is an extended version of the Django tutorial project with:
- A statistics page (/polls/stats/)
- An "is_open" flag to close polls from the admin
- A record of voters to disable voting multiple times
- An updated results page with piecharts (/polls/<question_id>/results)
